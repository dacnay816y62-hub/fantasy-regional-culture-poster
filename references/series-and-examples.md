## Batch workflow

1. List all regions.
2. Build a candidate pool for every region.
3. Generate 3-5 directions per region.
4. Compare the entire set before choosing launch directions.
5. Lock one analysis card per region.
6. Render or prompt the set.
7. Apply one shared typography system while preserving local differences.
8. Run a final global QC pass.

## Deduplication matrix

Track, at minimum:

| Region | Primary structure | Character logic | Composition | Dominant palette | Old/new relation | Epigraph tone |
|---|---|---|---|---|---|---|

Do not let adjacent posters repeat more than two of these dimensions. Rebuild a direction when its primary structure, deformation, and composition all duplicate an existing entry.

For repeated runs of the same region, avoid the previous primary structure, deformation method, palette strategy, old/new relationship, and sentence cadence.

## Example: 北京

**本次主题：** 中轴之城  
**主大字：** 京  
**主文化结构：** 中轴线、宫殿屋脊、城门秩序  
**辅助文化元素：** 红墙、门洞、长阴影  
**精神气质：** 秩序、边界、庄重、时间  
**色彩建议：** 宫墙红、暖金、深影黑、米白  
**构图建议：** 纵向轴线构图，大字压于建筑局部之上，强调门洞与留白  
**关键词：** 红墙｜中轴｜城阙｜院落｜秋声  
**一句短句：** 一条轴线，定义一座城的方向。  
**大字变形逻辑：** “京”字纵向拉长，内部负形借门洞关系，顶部横笔借屋檐轮廓  
**应避免内容：** 故宫全景、长城拼贴、旅游宣传语、符号堆砌

## Example: 广东

**本次主题：** 骑楼与江风  
**主大字：** 粤  
**主文化结构：** 骑楼拱廊、锅耳墙、珠江流线  
**辅助文化元素：** 老城砖墙与现代城市轮廓  
**精神气质：** 湿热、生长、开放、烟火  
**色彩建议：** 灰蓝、暖灰、深绿、潮湿砖色  
**构图建议：** 横向展开，裁切连续骑楼结构，少量叠入现代城市轮廓  
**关键词：** 江海｜骑楼｜湿热｜烟火｜生长  
**一句短句：** 江水经过旧城，也把新的生活带向更远的地方。  
**大字变形逻辑：** “粤”字外轮廓借锅耳墙，内部结构借骑楼拱券节奏  
**应避免内容：** 只放广州塔、俗艳岭南装饰、元素过满

## Example: 福建

**本次主题：** 海风入闽  
**主大字：** 闽  
**主文化结构：** 红砖古厝、燕尾脊、山海关系  
**辅助文化元素：** 海风、港口、巷道  
**精神气质：** 流动、含蓄、海洋性、日常感  
**色彩建议：** 红砖色、海雾灰、石灰白、蓝灰  
**构图建议：** 用局部屋脊和海风式留白，让大字与建筑轮廓穿插  
**关键词：** 红砖｜海风｜古厝｜山海｜巷陌  
**一句短句：** 风从海上来，也停在每一处屋檐之下。  
**大字变形逻辑：** “闽”字外框借院落边界，内部结构融入燕尾脊走势  
**应避免内容：** 土楼全景、海边明信片感、符号堆砌
